C#

The Virtual Pet Project was creating an C#-based console application that would allow a user to create and then talk to a type of animal. My application let the user set a name, age, and color of their choice and then prompted them to either speak, play, or feed their new monster. Written in C#, used loops and exit statements, and OOP.

Design

This project was particularly enjoyable for me to go through: we were given a screenshot of a website and then asked to used only HTML, CSS, and Bootstrap to recreate it. The real work was in teaching myself how to set up each Bootstrap box and column to center the content and keep as close to the original website as possible. Once I had the first few layouts figured out, I had a great time bringing the site to life and creating a page that would represent the brand.

Database

To create the Outfits App, I first made a relational database in SQL Server that would contain the description, style, and link for a photo for each clothing article. Then in Visual Studio, I linked the databases and used MVC to create a web app that would list each item in a gallery view, with the option to view more details. This was a great project for me because I enjoyed using SQL to learn how databases can impact the back-end of a website, and also gave me an opportinity to explore web apps with Microsoft products.
